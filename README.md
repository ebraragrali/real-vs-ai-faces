# Real vs AI-Generated Face Classification

This project classifies facial images as real or AI-generated.

## Dataset
140K Real and Fake Faces (Kaggle):
https://www.kaggle.com/datasets/xhlulu/140k-real-and-fake-faces

## Models
- SimpleCNN (from scratch)
- ResNet18 (transfer learning)

## Results
SimpleCNN achieved ~87% test accuracy.
ResNet18 achieved ~74% test accuracy.
